# Superstore-Sales-Performance-Dashboard
A Power BI sales analysis dashboard that visualizes sales, profit, discounts, and customer segments across regions and product categories to identify key trends, top performers, and loss-making areas using interactive filters.

# Tools Used
Power BI
Sample Superstore Dataset

# Objective
To demonstrate Practical DATA ANALYSIS & Visualization skills using Power BI And Deliver
business-ready insights through an interactive dashboard.

# Key Insights:-
- Profit analysis by product category & sub-category.
- Regional performance comparison.
- Impact of discounts and shipping modes.
- Customer segmentation analysis.
- Identification of top-perfomance and loss-making products.
